---
title: analyze()
description: The core diagnosis function that runs signals, detectors, and optional explanation/judge layers.
---

## Signature

```python
debugai.analyze(
    prompt: str,
    output: str,
    *,
    system_prompt: str = "",
    chunks: list[str] | None = None,
    similarity_scores: list[float] | None = None,
    retrieval_query: str | None = None,
    expected_output: str | None = None,
    model_name: str | None = None,
    temperature: float | None = None,
    max_tokens: int | None = None,
    context_window: int | None = None,
    latency_ms: int | None = None,
    token_usage: dict[str, int] | None = None,
    tool_calls: list[dict] | None = None,
    tools_expected: list[str] | None = None,
    response_schema: dict | None = None,
    thresholds: Thresholds = DEFAULT_THRESHOLDS,
    explain_with_llm: bool = True,
    lazy: bool = False,
    judge: bool = False,
    judge_model: str | None = None,
    openai_api_key: str | None = None,
    anthropic_api_key: str | None = None,
    variance_rerun: Callable | None = None,
    variance_runs: int = 3,
) -> dict
```

## Required fields

<ParamField path="prompt" type="string" required>
  The user prompt or query that produced the response.
</ParamField>

<ParamField path="output" type="string" required>
  The LLM response to diagnose.
</ParamField>

## Context fields

<ParamField path="system_prompt" type="string">
  System/developer rules. Enables citation checks, instruction judging, and
  better fix guidance.
</ParamField>

<ParamField path="chunks" type="list[str]">
  Retrieved documents or context chunks used for RAG grounding.
</ParamField>

<ParamField path="similarity_scores" type="list[float]">
  Retriever scores aligned with `chunks`. If omitted and chunks are present,
  DebugAI attempts to recompute similarity locally.
</ParamField>

<ParamField path="retrieval_query" type="string">
  Query sent to the retriever. Use this when a query rewrite differs from the
  user prompt; it enables query-drift detection.
</ParamField>

<ParamField path="expected_output" type="string">
  Optional reference answer attached to the capture record.
</ParamField>

## Runtime and contract fields

<ParamField path="model_name" type="string">
  Model identifier stored in the diagnosis metadata.
</ParamField>

<ParamField path="temperature" type="float">
  Sampling temperature; contributes to prompt-brittleness signals.
</ParamField>

<ParamField path="max_tokens" type="int">
  Requested output budget.
</ParamField>

<ParamField path="context_window" type="int">
  Model context limit; enables context-overflow detection.
</ParamField>

<ParamField path="latency_ms" type="int">
  End-to-end request latency.
</ParamField>

<ParamField path="token_usage" type="dict">
  Token counts, usually `{"prompt": n, "completion": n, "total": n}`.
</ParamField>

<ParamField path="tool_calls" type="list[dict]">
  Actual tool/function calls returned by the model. Include `name`, `input`,
  `output` or `result`, `status`, and `error` when available.
</ParamField>

<ParamField path="tools_expected" type="list[str]">
  Tool/function names the model was expected to call.
</ParamField>

<ParamField path="response_schema" type="dict">
  JSON Schema used to detect structured-output violations.
</ParamField>

## Optional LLM layers

<ParamField path="explain_with_llm" type="bool" default="true">
  Run the optional LLM explainer. Set to `False` for fully deterministic local
  diagnosis.
</ParamField>

<ParamField path="judge" type="bool" default="false">
  Run the instruction-adherence judge against `system_prompt`. Uses an LLM when
  configured and falls back to deterministic heuristics when no key is present.
</ParamField>

<ParamField path="variance_rerun" type="callable">
  Callable used to re-run the model for measured variance instead of the proxy
  estimate.
</ParamField>

## Examples

<CodeGroup>
```python Minimal
from debugai import analyze

result = analyze(prompt="...", output="...", explain_with_llm=False)
print(result["healthy"])
print(result["primary"])
```

```python RAG grounding
result = analyze(
    prompt="What is the cancellation policy?",
    output="You can cancel anytime for a full refund.",
    chunks=retrieved_docs,
    similarity_scores=scores,
    retrieval_query="cancellation refund policy",
    temperature=0.7,
    context_window=8192,
    explain_with_llm=False,
)
```

```python Tool contract
result = analyze(
    prompt="Search current shipping cutoff.",
    output="The cutoff is 5pm today.",
    tools_expected=["search"],
    tool_calls=[],
    explain_with_llm=False,
)
```

```python Schema validation
result = analyze(
    prompt="Classify this ticket.",
    output='{"status":"maybe"}',
    response_schema={
        "type": "object",
        "required": ["status", "answer"],
        "properties": {
            "status": {"type": "string", "enum": ["ok", "error"]},
            "answer": {"type": "string"}
        }
    },
    explain_with_llm=False,
)
```
