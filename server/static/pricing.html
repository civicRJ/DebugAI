Common questions

Do I need an API key?

The core diagnosis engine (8 signals, 5 detectors) is fully deterministic — no LLM, no API key required. LLM features (instruction judge, explainer, fix re-runs) use your own OpenAI or Anthropic key, set in your Account settings.

How does the free tier work?

500 diagnoses per calendar month. No credit card required. The counter resets on the 1st of each month. You can see your usage in the dashboard.

What counts as a diagnosis?

One call to analyze() or one request through wrap_llm() that runs the full signal + detector pipeline. Healthy requests still count. Playground previews do not count.

Can I self-host?

Yes. The full stack is open source on GitHub. Run docker compose up — PostgreSQL + Caddy auto-TLS included. Self-hosted is free forever.
